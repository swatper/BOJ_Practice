# BOJ_Practice

백준 온라인 저지(BOJ) 문제 풀이를 기록하는 저장소입니다.
[BaekjoonHub](https://github.com/BaekjoonHub/BaekjoonHub)를 통해 자동으로 푸시됩니다.

## 🛠 자동화 파이프라인 (Automation)
단순히 코드만 저장하는 것이 아니라, 학습한 내용을 체계적으로 관리하기 위해 다음과 같은 자동화 워크플로우를 직접 구현했습니다.

1. **GitHub Actions**: 코드 푸시 발생 시 파이썬 스크립트 자동 실행
2. **README Parsing**: 문제 번호, 제목, 티어, 알고리즘 분류 자동 추출
3. **Notion API**: 추출된 데이터를 기반으로 노션 데이터베이스에 학습 내용 저장
